# 📰 AI Fake News and Potential Scam Detection System

An AI-powered web application developed as part of the **IBM SkillsBuild AI Internship in collaboration with Edunet Foundation**. The system helps users identify fake news and detect potential online scams using Google's Gemini AI, providing intelligent analysis and explanations to promote digital awareness and online safety.

---

## 🚀 Features

- 📰 AI-based Fake News Detection
- ⚠️ Potential Scam Detection
- 🤖 Powered by Google Gemini AI
- 💬 Detailed AI-generated explanations
- 💻 Simple and user-friendly interface
- ⚡ Fast and accurate text analysis

---

## 🛠️ Technologies Used

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Uvraj-singh/Fake-News-and-Potential-Scam-Detection-System.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create an API key file

Create a file named `api_key.env`

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the application

```bash
streamlit run fake_news.py
```

---

## 📖 How It Works

1. Enter a news article or suspicious message.
2. Select the appropriate analysis option.
3. The application sends the text to Google's Gemini AI.
4. Gemini analyzes the content.
5. The application displays whether the content appears genuine or potentially misleading, along with an explanation.
